cmake_exmample
==============

A sample building cpp using cmake, a simple folder structure, and gmock/gtest.

Rebuilt using guidance from [Modern CMake](https://gitlab.com/CLIUtils/modern-cmake)

```shell
mkdir build
cd build
cmake ..
make all
make test
```
